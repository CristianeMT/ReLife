🦊 ReLife

ReLife é um jogo educacional 2D desenvolvido no **Construct 2**, com o objetivo de conscientizar jogadores sobre os impactos da poluição, queimadas e desmatamento — e o poder que cada um tem de restaurar o meio ambiente.


🕹️ Controles

| Tecla | Ação         |
|-------|--------------|
| `W`   | Mover para cima |
| `A`   | Mover para a esquerda |
| `S`   | Mover para baixo |
| `D`   | Mover para a direita |


🌱 Temas Abordados

- ♻️ Poluição e seus impactos no ecossistema
- 🔥 Queimadas e destruição de habitats
- 🌳 Desmatamento e perda da biodiversidade
- 🌿 Restauração ambiental e sustentabilidade

---

🛠️ Tecnologias Utilizadas

| [Construct 2](https://www.construct.net/) | Engine de desenvolvimento do jogo |
| HTML5 / JavaScript | Tecnologia base gerada pelo Construct 2 |



🚀 Como Jogar

1. Acesse o link do jogo https://www.construct.net/en/free-online-games/relife-86935/play
2. Aguarde o carregamento no navegador
3. Use **W, A, S, D** para movimentar a raposa
4. Siga as missões e ajude a restaurar o meio ambiente!
